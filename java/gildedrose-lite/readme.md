# Gilded Rose starting position in Java

# Prerequisites
- Maven
- Java 17 or higher
- An IDE such as Intelij or vscode

# Setup
This project may be initialised by running the command
`mvn clean install`

## Intro
GildedRose is an inventory system that updates daily.
It contains special items (Brie, Backstage Passes, Sulfuras, Conjured), and normal items (everything else).

All Items contain the properties:
`SellIn`: Represents the number of days left to sell an item.
`Quality`: Represents how valuable an item is.

| Item Type      | SellIn Change | Quality Change                                            |
| -------------- | ------------- |-----------------------------------------------------------|
| Normal Item    | -1 daily      | -1 per day (before sell-by), -2 per day (after)  (min 0)  |
| Aged Brie      | -1 daily      | +1 per day (before sell-by), +2 per day (after)  (max 50) |
| Backstage Pass | -1 daily      | +1 (>10 days), +2 (10-6 days), +3 (5-1 days), 0 (>1days)  |
| Sulfuras       | No Change     | No Change (stays 80)                                      |

## Your task
The inventory system has been 'vibe coded' by an overly eager project manager, and it is
your task to refactor it.

You are expected to (in any order):

- Refactor and improve the `GildedRose` class in any way you see fit
- Implement unit tests in the `UnitTests` class for the inventory system

You may not:
- Refactor or alter the `Item` class
- Refactor or alter any `BaselineFixture` or `VerifyOutput`

## Unit tests
The following unit tests may be useful:

- Quality & SellIn for normal items
- Quality & SellIn for special items
- Quality once the sell-in date has passed
- The Quality of Brie
- The Quality of backstage passes

## Running unit tests / Verifying the application output
Unit tests can be run via your IDE, or by using the following command
`mvn test -Dtest=UnitTests`

You can verify any changes that you have made to the `GildedRose` class by running `VerifyOutput`.
`mvn test -Dtest=VerifyOutput`

This test will compare the generated SellIn and Quality generated by the `GildedRose` class against a static text file.
